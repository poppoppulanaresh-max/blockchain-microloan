This platform enables decentralized micro-loans using blockchain.

\- React

\- Web3.js

\- MetaMask



Backend

\- Node.js

\- Express.js

\- JWT

\- MySQL



Blockchain

\- Solidity Smart Contracts

\- Ethereum Sepolia Testnet
